# Simple Password Generator 🔐

Μια απλή και αποδοτική εφαρμογή σε **Python** για τη δημιουργία τυχαίων και ισχυρών κωδικών πρόσβασης. Το εργαλείο αυτό είναι ιδανικό για όσους αναζητούν έναν γρήγορο τρόπο παραγωγής κωδικών χωρίς την ανάγκη εξωτερικών βιβλιοθηκών.

## 🚀 Χαρακτηριστικά
- **Πλήρης συνδυασμός χαρακτήρων:** Περιλαμβάνει πεζά, κεφαλαία, αριθμούς και σύμβολα.
- **Δυναμικό μήκος:** Ο χρήστης ορίζει πόσους χαρακτήρες επιθυμεί.
- **Minimalist Code:** Καθαρός κώδικας, εύκολος στην κατανόηση και την παραμετροποίηση.

## 🛠️ Τεχνολογίες
Το project χρησιμοποιεί αποκλειστικά τις ενσωματωμένες βιβλιοθήκες της Python:
- `string`: Για την πρόσβαση σε έτοιμα σύνολα χαρακτήρων.
- `random`: Για την τυχαία επιλογή χαρακτήρων.
