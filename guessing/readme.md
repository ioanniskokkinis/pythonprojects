# 🎯 Number Guessing Game (Binary Search Logic)

![Python Version](https://img.shields.io/badge/python-3.x-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Ένα διαδραστικό παιχνίδι πρόβλεψης αριθμών γραμμένο σε **Python**. Το παιχνίδι δεν ορίζει τυχαία τις προσπάθειες του παίκτη, αλλά τις υπολογίζει δυναμικά χρησιμοποιώντας τη λογική της **Δυαδικής Αναζήτησης**.

## 🚀 Χαρακτηριστικά

*   **Δυναμικό Εύρος:** Ο παίκτης επιλέγει το ξεκίνημα και το τέλος του εύρους των αριθμών.
*   **Έξυπνος Υπολογισμός Προσπαθειών:** Το παιχνίδι υπολογίζει τον ελάχιστο αριθμό κινήσεων που απαιτούνται μαθηματικά για να βρεθεί ο αριθμός, χρησιμοποιώντας τον τύπο:  
    $$log_2(\text{range\_size})$$
*   **Error Handling:** Προστασία από λανθασμένες εισαγωγές (γράμματα αντί για αριθμούς).
*   **Real-time Feedback:** Καθοδήγηση του παίκτη αν ο αριθμός είναι μεγαλύτερος ή μικρότερος.

## 🛠️ Πώς Λειτουργεί

Το παιχνίδι βασίζεται στην αρχή του **Binary Search**. Για οποιοδήποτε εύρος αριθμών $N$, ο μέγιστος αριθμός προσπαθειών που χρειάζεται ένας παίκτης για να βρει το αποτέλεσμα (αν παίζει τέλεια) είναι $\lceil \log_2(N) \rceil$. 

Παράδειγμα:
- Για εύρος 1-100: **7 προσπάθειες**
- Για εύρος 1-1000: **10 προσπάθειες**
