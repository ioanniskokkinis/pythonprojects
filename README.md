# My Python Lab 🐍

Hey there! This is where I’m keeping my first Python scripts. I’m building these to get comfortable with the basics and actually make some useful tools while I'm at it.

## 🛠️ What's inside?

### 1. Task Master (To-Do List)
A simple CLI tool to keep track of stuff I need to do.
* **How it works:** It uses a Python list to store tasks and a `while` loop to keep the menu running.
* **The Struggle:** My biggest "aha!" moment here was realizing that Python lists start at 0, not 1. Classic beginner mistake!

### 2. PassGen (Secure Password Generator)
Since "123456" is a terrible password, I made this to generate random, strong strings.
* **Tech Stack:** I used the `secrets` module instead of `random` because it's much more secure for passwords.
* **Logic:** It mixes letters, digits, and punctuation into one string.
