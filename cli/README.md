# 💰 Personal Expense Tracker (CLI)

A lightweight Command Line Interface (CLI) tool built with Python to help you track your daily spending, categorize expenses, and stay within your budget.

## 🚀 Features
- **Add Expenses:** Quickly log spending with a description, category, and amount.
- **Auto-Timestamp:** Automatically records the date of every transaction.
- **CSV Storage:** Data is saved in a structured `expenses.csv` file, compatible with Excel and Google Sheets.
- **Spending Summary:** View your total spending or filter by category (Food, Transport, Rent, etc.).

## 🛠️ Tech Stack
- **Python 3.x**
- **CSV Module:** For persistent data storage.
- **Datetime Library:** For chronological tracking.

## 📂 Project Structure
- `tracker.py`: The main application logic.
- `expenses.csv`: The database (generated automatically on first run).
- `utils.py`: (Optional) Helper functions for calculations.
