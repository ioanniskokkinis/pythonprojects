# Rock, Paper, Scissors (Python)

A lightweight, command-line implementation of the classic game. This version is designed to be quick to run and easy to understand, featuring a persistent game loop and a real-time score tracker.

---

## 🚀 Features
* **Continuous Play:** Uses a `while` loop so you can play multiple rounds in one session.
* **Score Tracking:** Maintains a running tally of wins for both the user and the computer.
* **Case Insensitivity:** Accepts inputs like "Rock", "ROCK", or "rock" seamlessly.
* **Clean Exit:** Displays your final results when you choose to quit.

---

## 🛠️ Installation & Usage

1. **Clone or Copy** the code into a file named `rps_game.py`.
2. **Open your terminal** or command prompt in the directory where the file is saved.
3. **Run the game** using:
   ```bash
   python rps_game.py
