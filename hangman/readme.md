# Python Word Guessing Game

Ένα απλό και διασκεδαστικό παιχνίδι εύρεσης λέξεων (Hangman-style) σε περιβάλλον Command Line. Το project αυτό αποτελεί μέρος μιας σειράς από CLI εργαλεία γραμμένα σε Python.

## 🚀 Χαρακτηριστικά
- Τυχαία επιλογή λέξεων από προκαθορισμένη λίστα.
- Σύστημα περιορισμένων προσπαθειών (6 ζωές).
- Διαδραστικό περιβάλλον χρήστη μέσω τερματικού.

## 🛠 Εγκατάσταση & Χρήση

### Τοπική Εκτέλεση
1. Βεβαιωθείτε ότι έχετε εγκατεστημένη την **Python 3.x**.
2. Κατεβάστε το αρχείο στον υπολογιστή σας.
3. Τρέξτε την εντολή:
   ```bash
   python main.py
